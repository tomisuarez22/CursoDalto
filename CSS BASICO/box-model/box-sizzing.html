Propiedad Box Sizzing

La propiedad Box Sizzing define como se debe expandir un elemento en bloque según el valor que se le asigne a la propiedad. Por defecto, la propiedad de "box-sizzing" tiene el valor de content-box.

Valores de la propiedad box-sizzing

  • content-box

    • Es el valor que tienen por defecto los elementos en bloque.

  • border-box

    • Es el valor que permite al bloque mantener su tamaño definido (width y height) a pesar de tener padding, margin y border.

Ejemplo div con propiedad content-box

Propiedades:
background-color: cadetblue;
width: 150px;
height: 150px;
margin: auto;
padding: 20px;
border: 10px black solid;
Como se observa, el div es de 150x150px definidamente pero al tener padding, border y margin este se le suman los px a su tamaño definido y no terminamos de saber cual es el tamaño real del div.

Ejemplo div con propiedad border-box

Propiedades:
background-color: cadetblue;
width: 150px;
height: 150px;
margin: auto;
padding: 20px;
border: 10px black solid;
Ahora como se observa, el div se mantiene en 150x150px gracias la propiedad box-sizzing: border-box. Esto nos facilita los cálculos de tamaño.
