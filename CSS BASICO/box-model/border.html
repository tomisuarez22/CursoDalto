Bordes

La propiedad "border" nos permite definir bordes en los elementos.

Propiedades de border:

  • border-width

    • Define el tamaño del borde.

  • border-color

    • Define el color del borde.

  • border-style

    • Define el estilo del borde. (Puede ser sólido, transparente, linea de puntos).

  • border-radius

    • Define el redondeo del borde.

Podemos hacer "shorthand" con las propiedades width - color - style utilizando solo la palabra "border".
Ejemplo:
border: 3px red solid

Ejemplo

Este div esta definido con las siguientes propiedades de border:
