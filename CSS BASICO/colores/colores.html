Colores

Los colores en CSS pueden ser seleccionados de distintas maneras, pero principalmente se basa en RGB, HEX, HSL.

EJEMPLO

Clasificación:

  • RGB

    • El formato en CSS sería (R, G, B) siendo sus valores del 0 al 255. En este caso el color del div sería (255, 204, 204)

  • HEX

    • El formato en CSS sería #RRGGBB o #RGB siendo sus valores del 0 a la F. En este caso el color del div sería #FFCCCC o #FCC

  • HSL

    • El formato en CSS sería hsl(Matiz, Saturación, Luminosidad) donde la matíz representa el tipo de color en si mismo y va de 0° al 360° en una ruleta de colores. La saturación representa la pureza del color que va del 0% al 100% y la Luminosidad representa la cantidad de luz que refleja el color y que tambien sus posibles valores van del 0% al 100%. En este caso el color del div sería hsl(0, 100%, 90%).
