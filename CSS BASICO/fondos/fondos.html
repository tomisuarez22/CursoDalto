Fondos, Gradientes y Sombras

CSS nos permite agregar y personalizar distintos fondos para nuestra página web. Podemos agregar imágenes, degradados e incluso sombras a distintos elementos.

Existen distintas propiedades tipo "background" para personalizar los fondos.

Propiedades:

  • background-color

    • Define el color de fondo de un elemento. Puede ser un color o la palabra clave "transparent".

  • background-image

    • Establece una o más imagenes de fondo para un elemento.

  • background-size

    • Especifica el tamaño de las imagenes de fondo.

  • background-position

    • Define la posición inicial de la imagen de fondo especificada.

  • background-repeat

    • Define como se repiten los fondos del documento. Puede ser repetido sobre el eje X, Y, ambos ejes o puede no ser repetido.

  • background-attachment

    • Determina si la posición de la imagen será fija dentro de la pantalla o se desplazará con su bloque contenedor. Sus posibles valores son:

    • scroll

      • Es el valor predeterminado, se moverá dentro de la pantalla junto al bloque que la contiene.

    • fixed

      • No se moverá dentro de la pantalla junto al bloque que la contiene.

Propiedad "background" como "shorthand"

Podemos utilizar como "shorthand" la propiedad background. De manera que su sintáxis sería:
background: url(imagen) background-position background-size background-repeat background-attachment background-color

Gradientes

Gradiente lineal

Mediante la funcion "linear-gradient(dirección de gradiente, color inicio, color final)" podemos agregar un degradado desde un color hacia otro.

Texto

background: linear-gradient(transparent, #f009);

Podemos agregarle más de un color a la función si queremos.

Gradiente radial

Mediante la función "radial-gradient(color inicio, color final)" podemos agregar un degradado de forma RADIAL.
