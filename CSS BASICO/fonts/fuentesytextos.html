Tipografías e importación de fuentes externas

Podemos importar fuentes de manera externa en caso de que querramos utilizar una fuente terciarizada. Para ello hay 2 formas comunes para hacerlo.

  1. Importando el código HTML

    2. Debemos de copiar el código HTML de la fuente y pegarlo en el head de nuestro archivo HTML y asi podemos utilizarla luego en CSS como una fuente predeterminada.

    "Este es un texto con una fuente importada mediante el código HTML en el head de mi archivo :D."

  2. Creando un archivo CSS y asignar la tipografía manualmente

    3. Debemos descargar la fuente y sus tipografías, importarlas luego en un archivo CSS aparte y asignarles su nombre y tipografía correspondiente de manera manual para luego poder utilizarlas de manera normal en nuestro archivo style.css

    "Este es un texto con una fuente importada mediante un archivo de CSS aparte donde le asigné yo mismo la tipografía correspondiente y específica."
