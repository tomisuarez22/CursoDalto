Información Básica

Labels

Un label es una etiqueta que sirve para detallar el título o los datos que se tienen que ingresar en un input.

TextArea y sus propiedades

La etiqueta textarea es un input personalizable que permite ingresar cualquier tipo de dato en un espacio agrandado.

Propiedades de textarea en CSS

  • Resize

    • Sirve para definir en que posición desplazar el textarea. Puede ser horizontal, vertical o que no se desplace.

Posibles atributos de textarea en HTML

  • Placeholder

    • Detalla un texto transparente para que sirva como guía.

  • Readonly

    • Al definir este booleano, le sacamos al usuario el permiso de modificar el textarea.

  • Disabled

    • Al definir este booleano, le sacamos al usuario el permiso de modificar el textarea y directamente no se puede seleccionar ni tampoco se cuenta a la hora de enviar al servidor.

  • Max-length

    • Este atributo permite definir un máximo de caracteres.

  • Row

    • Este atributo define la cantidad de lineas de texto que va a tener de forma inicial el textarea.

Select, datalist y options

Select

La etiqueta select nos permite crear una lista selectiva para el usuario conectada con distintos ítems mediante la etiqueta option.

Datalist

La etiqueta datalist nos permite crear una lista selectiva a través de entrada de texto para el usuario conectada con distintos resultados mediante la etiqueta option.
Las etiquetas datalist tienen que ser agregada dentro de una etiqueta input cuyo atributo obligatorio debe ser list.

Ejemplo

La etiqueta datalist debe estar conectada mediante su atributo id con el atributo list de el input.
