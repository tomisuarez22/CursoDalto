Iconos

Para colocar un ícono a nuestra página web debemos linkearlo en nuestro "head" mediante la etiqueta:
"link rel="icon href="rutaIcono" type="image/png". El ícono debe ser de 32x32px o 64x64px.

Metadatos

Son etiquetas cuya finalidad es proporcionar información interna sobre nuestra web. Estas etiquetas están ubicadas en el head y pueden tener atributos como "name" y content" donde:

  • El atributo name especifica el tipo de metadato del que se trata.
  • El atributo content especifica el contenido del metadato en sí.

Tipos de metadatos

  • meta charset

    • Este tipo de metadato es para la codificación de caracteres de nuestra página web. Donde generalmente utilizamos el tipo "UTF-8" ya que este codificador contiene todos los caracteres de todos los idiomas.

  • meta name="description"

    • Define la descripción del contenido de la página beneficiando al motor de búsqueda. La descripción la detallamos en el atributo content.

  • meta name="keywords"

    • Define las palabras claves de búsqueda sobre la página web. Las palabras las detallamos en el atributo content.

  • meta name="author"

    • Define el autor de la página web. Detallamos el nombre en el atributo content.

  • meta name="robots"

    • Sirve para definir el comportamiento de los motores de búsqueda según el valor que definamos en content.

      Posibles valores

    • nosnippet
    • noarchive
    • noimageindex

    Metatags OG

  • OG:TITLE
  • OG:IMAGE
  • OG:DESCRIPTION

    • NOTA: En la página metatags.io podemos generar el código para modificar los OG a nuestro gusto de una manera más cómoda..

Lazy Loading

Este es un tipo de propiedad que especifíca que elementos priorizar para cargar según lo que el usuario vea en la página.

Accesibilidad WEB

Este tema abarca la inclusión a diferentes personas con distintas discapacidades que no les permite disfrutar la página a su 100%. Como son las personas ciegas, daltónicas, sordas, etc. Una página web debe ser realizada con completa accesibilidad a estas personas.

Mediante el atributo "alt" y "aria-label" podemos aclarar ciertas etiquetas para que los narradores lo reconozcan y describan al usuario la etiqueta.
