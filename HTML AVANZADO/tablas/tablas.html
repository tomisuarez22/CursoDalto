Tablas

Las tablas nos permiten mostrar datos tabulares. Estas se crean mediante la etiqueta "table". Y para agregar filas lo hacemos mediante la etiqueta "tr" y le asignamos columnas mediante la etiqueta "td".

EJEMPLO

Nombre Apellido Edad
Tomás Suarez 20
Priscila Vergara 19
Norma Leon 67

Las tablas pueden tener atributos para combinar filas y columnas como es "coldspan" y el valor que ocupa sería el de la cantidad de espacios que se desea que ocupe.

Semántica en las tablas

Su semántica se utiliza a partir de las siguientes etiquetas:

  • thead

    • Dentro de esta etiqueta va el encabezado de la tabla.

  • tbody

    • Dentro de esta etiqueta va el cuerpo de la tabla.

  • tfoot

    • Dentro de esta etiqueta va el pie de la tabla.
