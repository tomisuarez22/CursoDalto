Enlaces Avanzado

Como sabemos, la etiqueta a permite redirigir a los usuarios a distintas páginas y sectores de la web. Pero tambien tiene otras funcionalidades.

  • Dirigirse a una parte de la misma ventana.

    • Mediante una ID podemos dirigirnos a una parte de la misma ventana. Solo debemos enlazar el ID de la etiqueta que querramos dirigir a el atributo href de la etiqueta a.

  • Descargar un archivo que queramos

    • Mediante el atributo booleano "download" podemos descargar el archivo que esté enlazado en el atributo href.

  • Mandar un mail

    • Mediante el atributo "mailto:correo@gmail.com" nos abre automáticamente nuestra aplicación para enviar correos electrónicos a la dirección detallada.

  • Llamar a un telefono determinado

    • Mediante el atributo "tel:+54123456789" nos abre automáticamente una aplicación de llamada para llamar a el numero telefónico seleccionado.

Seguridad en nuestra página web

Mediante el atributo "rel" podemos definir que tipo de modificaciones y datos de nuestra página web permitimos al usuario. Con distintos valores podemos restringirle el acceso al usuario.

Valores para restringir distintos accesos en las etiquetas "a".

  • rel="noreferrer"

    • Este atributo evita pasar la información de referencia al sitio web destino eliminando la información de referencia del encabezado HTTP.

  • rel="nofollow"

    • Este atributo le indica a los motores de búsqueda que no pasen ningún PageRank de una página a otra.

  • rel="noopener"

    • Este atributo impide que la página de apertura obtenga cualquier tipo de acceso a la página principal.
