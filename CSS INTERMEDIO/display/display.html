Display en CSS

Los elementos de HTML y su comportamiento

Los elementos en HTML tienen un comportamiento predeterminado según el elemento que sea. Este comportamiento nos dice que tipo de elemento es el que estamos utilizando. Entre los tipos de elementos podemos destacar:

  • Elementos en bloque
  • Elementos en línea

¿Qué es la propiedad Display en CSS?

La propiedad display en CSS es una propiedad que define el comportamiento de los elementos mediante sus distintos valores que puede adoptar. Entre ellos están:

  • Block

    • Define al elemento como un bloque, es decir ocupa todo el ancho disponible y su alto dependerá de su contenido. Estos elementos se apilan entre sí.

  • In-line

    • Define al elemento como un elemento en línea, es decir que se comporta como una etiqueta donde su ancho y alto dependen únicamente del contenido y no podemos darle margin ni padding. Estos elementos normalmente se juntan entre sí.

  • In-line block

    • Define al elemento como un elemento en bloque pero esta vez su ancho y alto dependen de su contenido, es decir que predeterminadamente NO va a ocupar el ancho de la página web y pueden juntarse entre sí.

  • None

    • Este valor define al elemento nulo, es decir que no ocuparía ni espacio en la página ni tampoco se mostraría.
