Overflow y sus propiedades

Overflow nos proporciona distintos valores y propiedades para definir el comportamiento de los contenidos dentro de los contenedores.

  • overflow-x

    • Especifica y modifica el comportamiento en el eje x.

  • overflow-y

    • Especifica y modifica el comportamiento en el eje y.

Posibles valores de overflow

  • visible

    • Valor por defecto. El contenido no es recortado, podría ser dibujado fuera de la caja contenedora.

  • hidden

    • El contenido es recortado y no se muestran barras de posición.

  • scroll

    • El contenido es recortado y el navegador web usa las barras de desplazamiento, se haya recortado el contenido o no.

  • auto

    • Depende del agente de usuario. El navegador provee barras de desplazamiento si hay contenido excedente, en el caso de que no no pasa nada.

Control de flujo de texto: propiedad white-space

CSS nos permite modificar y controlar el flujo de texto de un parrafo. Permitiendo distintos saltos de línea entre otras cosas.

  • normal

    • Valor por defecto, las líneas se dividen según sea necesario para llenar los cuadros de línea.

  • nowrap

    • Elimina los saltos de línea.

  • pre

    • Los saltos de línea están definidos según como el desarrollador redactó el texto y permite que el texto se desborde.

  • pre-wrap

    • Los saltos de línea están definidos según como el desarrollador redactó el texto y NO permite que el texto se desborde.

  • break-spaces
