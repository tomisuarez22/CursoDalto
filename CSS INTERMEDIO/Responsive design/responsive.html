Responsive design

¿Qué es el responsive design?

El responsive design es un enfoque de diseño web que busca crear sitios web que proporcione al usuario una experiencia óptima y compatible con una amplia gama de tamaños de pantalla y dispositivos. El objetivo es asegurar que el contenido se vea bien y sea fácil de usar sin importar el dispositivo que el usuario esté utilizando.

Atributos Srcset y Sizes

El atributo srcset especifica que recurso utilizar dependiendo de la resolución que tenga el usuario.
El atributo sizes especifica una mediaquery, es decir a partir de que resolución que imagenes poner.

Sintáxis

srcset = "ubicaciónRecurso cantidadAnchow"

Ejemplo: < img src="imagenPredeterminada.png" srcset:"imagenAlternativa.png 300w">

sizes = "(max-width: 400px) 360px, (max-width: 600px) 600px, 900px"
Es decir, si el tamaño del ancho es hasta 400px que ponga la imagen de 360px, si la resolución esta entre 400px y 600px colocar la imagen de 600px, si no si es mayor colocar la de 900px.
