Herencia, cascada y especifidad

Herencia

La herencia en CSS define que algunos valores de las propiedades que se han establecidos para los elementos padre, los heredan los elementos hijo, mientras que los otros que no son familiares no.

Herencia: inherit e initial

CSS nos proporciona cuatro valores de propiedad universales para el CONTROL DE LA HERENCIA.

Valores de herencia

  • inherit

    • Este valor quiere decir que el elemento va a adaptar el valor de su elemento padre. (activa la herencia).

  • initial

    • Este valor define que no importa si hay un elemento padre con propiedades distintas, este no adapta los valores de su padre y queda con los valores por defecto del navegador.

  • unset

    • Restablece la propiedad a su valor natural, lo que significa que si la propiedad se hereda de forma natural, actúa como inherit, y en caso contrario como initial.

  • revert

    • Esta propiedad revierte el valor en cascada de la propiedad de su valor actual al valor que la propiedad habría tenido si el origen del estilo actual no hubiera realizado cambios en el elemento actual.

Especifidad

Define la prioridad de los selectores en CSS. Por ejemplo, si nosotros definimos la propiedad de un elemento seleccionando solo por elemento, y luego definimos otra propiedad a este elemento seleccionado pero por clase, el elemento quedaría con las propiedades asignadas en el elemento que seleccionamos por clase ya que el selector de clase es más específico.

CSS y su especifidad

Las distintas formas de modificar en CSS tienen diferentes prioridades. Van del 1 la más importante al 6 que es el menos importante.

  1. !important

    2. Esta declaración es la más prioritaria. Es decir al nosoros declarar esto luego de una regla, el elemento se va a modificar si o si, sin darle importancia a las demas modificaciones que se hayan realizado o a realizar.

  2. Atributo style=""

    3. Este atributo lo declaramos en HTML y dentro de el podemos asignar una propiedad al elemento.

  3. Etiqueta "style"

    4. Dentro de esta etiqueta en nuestro archivo HTML, podemos asignar las propiedades a los elementos que querramos.

  4. En nuestro archivo style.css

    5. Dentro de este archivo podemos asignar las propiedades a los elementos de HTML.

  5. Modificaciones que haga el usuario desde el navegador

    6. El usuario puede modificar el estilo de la página web a través de una extensión que le permita modificar el archivo CSS de la página.

  6. El navegador por predeterminado

    7. Estas son las modificaciones que ya hace por defecto el navegador y por supuesto las menos especificas.

Cascada en CSS

El concepto de cascada en CSS nos dice que el orden de las acciones que nosotros realicemos en CSS, importa. Es decir pueden aplicarse dos propiedades a un mismo elemento por separado y CSS va a tomar la que esté aplicada al último.
