Flexbox

¿Qué es Flexbox?

Flexbox significa "Caja flexible" el cual es un módulo diseñado como un modelo y método que pueda ayudar a distribuir el espacio entre los ítems de un contenedor y mejorar las capacidades de alineación.

Los dos ejes de Flexbox

Cuando trabajamos con Flexbox, actúan dos ejes. El eje principal y el eje cruzado, donde el eje principal está definido por la propiedad flex-direction, y el eje cruzado es perpendicular a este.

Propiedad flex-wrap

Permite controlar el comportamiento de los ítems cuando su contenedor ya no puede seguir mostrando los ítems por su tamaño. Sus posibles valores:

  • nowrap

    • Los elementos flex son distribuidos en una sola línea.

  • wrap

    • Los elementos flex son colocados en varias líneas.

  • wrap-reverse

Abreviación flex-flow

Es una abreviación de las propiedades flex-direction y flex-wrap.

Propiedad align-items

Controla la alineación de elementos en el eje cruzado. Esta propiedad se utiliza cuando hay solo UNA línea de elementos.

Valores

  • stretch

    • Si los elementos son más pequeños que el contenedor de alineación, estos se ampliarán igualmente para llenar el contenedor, respetando los límites de ancho y alto de los elementos.

  • start

    • Los artículos se colocan al ras entre sí hacia el borde inicial del contenedor.

  • center

    • Los artículos se colocan en el centro del contenedor.

  • end

    • Los artículos se colocan al ras entre sí hacia el borde final del contenedor.

Propiedad align-self

Esta propiedad se aplica a los elementos flexibles y cuando está aplicada la propiedad align-items. Especifica la ubicación del elemento dentro del contenedor.

Propiedad order

Esta propiedad nos permite definir un orden entre elementos flexibles.

Propiedad flex-grow

Esta propiedad permite a los elementos utilizar el espacio disponible del contenedor flexible para que estos aumenten su tamaño a medida que el contenedor lo aumenta. Mediante su valor definimos la cantidad de espacio disponible que puede ser utilizada por el elemento.

Propiedad flex-shrink

Define cuanto tamaño se va a encoger el elemento. Cuando el tamaño es menor, y no permitimos wrap, flexbox permite disminuir el ancho o alto de los elementos para que no se deformen. Esta disminución que sufren estos elementos podemos aumentarla o anularla mediante la propiedad flex-shrink.

Propiedad flex-basis

Definimos el tamaño base de un elemento. Cuando utilizamos flexbox esta propiedad reemplaza el min-width o min-heigth. Cuando los elementos sufren la disminución de su tamaño, la propiedad flex-basis establece el máximo de disminución y una vez que sobrepasa ese máximo debe de wrapearse.

Abreviación flex

Mediante la propiedad flex podemos abreviar las propiedades Shrink Grow Basis.

Propiedad gap

Esta propiedad define el espacio que va haber entre los elementos flexibles.
