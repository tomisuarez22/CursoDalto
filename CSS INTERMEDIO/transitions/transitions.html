Transiciones en CSS

Las transiciones en CSS nos permiten darle una mejor apariencia a nuestra página y experiencia para el usuario.

CSS

Propiedades de las transiciones

Existen distintas propiedades en CSS para definir y modificar nuestras transiciones.

  • transition-property

    • Especifica el nombre o nombres de las propiedades CSS a la que deberían aplicarse las transiciones.

  • transition-duration

    • Especifica la duración en la que sucederán las transiciones. Se puede especificar una única duración que se aplique a todas las propiedades durante la transición o valores múltiples que permitan a cada propiedad de transición un período de tiempo diferente.

  • transition-delay

    • Define el timepo de espera para dar inicio a la transición

  • transition-timing-function

    • Especifica la curva cúbica bézier que se usa para definir cómo se computan los valores intermedios para las propiedades.

    Posibles valores por defecto:

    • ease

      • Valor predeterminado, es lineal.

    • ease-in

      • Arranca lento y finaliza fuerte.

    • ease-in-out

      • Arranca lento y termina lento.

    • cubic-bezier

      • Define personalizadamente una aceleración y tiempo en la transición.

Abreviación de las transiciones

Podemos abreviar las propiedades transition-property, transition-duration y transition-timing-function con tan solo usando la propiedad transition. Es decir:

transition: propiedad duración timing-function
