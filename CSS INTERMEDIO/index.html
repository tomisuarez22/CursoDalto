Metodología BEM

La metodología BEM es 'Block Element Modify'. Está basada en una forma de programar en CSS Y HTML que nos sirve para reciclar código, evitar problemas de especifidad y hacer el código más ameno. Esta metodología define cómo deben ser nombradas las clases en HTML para posteriormente modificarlas con CSS de manera fácil, sencilla y clara.
