Selectores en CSS

Existen distintas formas de seleccionar un elemento en CSS.

Formas de selección en CSS

  • Selección de img por atributo

    • Podemos seleccionar los elementos según el atributo que posean. mediante el nombre del elemento y [atributo="valor"]. Ej: img[src="imagen.png"]{}".

    • Si nosotros colocamos img[src$="valor"]

      • Se seleccionan solo las imagenes que TERMINAN con ese valor. Por ejemplo si nosotros ponemos img[src$=".png"] este va a seleccionar solo las imágenes .png.

    • Si nosotros colocamos img[src^="valor"]

      • Se seleccionan solo las imagenes que COMIENZAN con ese valor. Por ejemplo si nosotros ponemos [src^="tomas"] este va a seleccionar solo las imágenes que comiencen con tomas.

  • Selección de hijo general

    • Podemos seleccionar elementos que estén dentro de otro elemento ya sea hijo, nieto, tataranieto, etc. Para hacerlo, basta con poner primero el nombre del elemento contenedor y luego el nombre del elemento que esté dentro de ese contenedor. Ejemplo: ul li{}
    Aquí se seleccionan todos los elementos li que estén dentro de el elemento ul.

  • Selección de hijo directo

    • Podemos seleccionar elementos que sean directamente hijos de un elemento. Para hacerlo, basta con poner primero el nombre del elemento padre seguido de un > y luego el nombre del elemento hijo. Ejemplo: ul > li{}.

  • Selección de hermano adyacente

    • Podemos seleccionar elementos hermanos. Es decir el próximo elemento que sigue de otro. Para hacerlo, debemos poner el primer elemento seguido de un +, y luego ponemos el segundo elemento que va a ser el que se seleccione. Ejemplo: h1 + p{}. Aqui va a seleccionar unicamente los elementos p que estén después de un elemento h1.

  • Selección por hermanos generales

    • Podemos seleccionar todos los elementos que vengan después de otro en específico. Para hacerlo, debemos poner el primer elemento seguido de un ~ y luego colocar el elemento que va a ser el que se seleccione. Ejemplo: h1 ~ p{}. Aquí va a seleccionar todos los elementos p que vayan después de un h1.
