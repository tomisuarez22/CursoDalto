Pseudo-clases

Son tipos de selectores de elementos en CSS, selecciona un estado específico de un elemento a través de palabras claves.

Tipos de Pseudo-clases

  • :hover

    • Selecciona y modifica un elemento cuando el cursor se le pasa por encima de este elemento.

    EJEMPLO

  • :active

    • Selecciona y modifica un elemento cuando le damos click a ese elemento.

    EJEMPLO

  • :first-child

    • Selecciona el primer elemento hijo de un elemento.

  • :last-child

    • Selecciona el último elemento hijo de un elemento.

  • :nth-child(numeroElemento)

    • Selecciona un elemento hijo de un elemento a elección. Si el valor del numero a seleccionar es 2n, 3n... etc... Selecciona uno si uno no, o dos si y dos no, etc.

  • :not(.clase)

    • Asigna las propiedades al elemento seleccionado menos a el elemento que tenga la clase que asignamos en ().

  • :empty

    • Selecciona al elemento y lo modifica en caso de que esté vacío.

  • :checked

    • Selecciona los checkbox que ya estén marcados.

  • :link

    • Selecciona los enlaces que aún no fueron visitados.

  • :visited

    • Selecciona los enlaces que ya fueron visitados.

  • :valid

    • Selecciona los inputs válidos.

  • :invalid

    • Selecciona los inputs inválidos.

  • :is()

    • Esta pseudo-clase selecciona determinados elementos hijos de un elemento padre.

  • :where()

    • Esta pseudo-clase selecciona determinados elementos hijos de un elemento padre pero con menos especifidad.

  • :has()

    • Esta pseudo-clase modifica a un elemento padre en caso de tener un determinado elemento hijo que colocaríamos dentro del ().
