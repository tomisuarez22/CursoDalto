Position en CSS

¿Que es la propiedad Position en CSS?

La propiedad position nos permite definir la posición de los elementos de HTML. Esta propiedad nos brinda una serie de propiedades extras:

  • TOP

    • Permite mover el elemento hacia abajo. (prioritaria)

  • LEFT

    • Permite mover el elemento hacia la derecha. (prioritaria)

  • RIGHT

    • Permite mover el elemento hacia la izquierda.

  • BOTTOM

    • Permite mover el elemento hacia arriba.

  • Z-INDEX

    • Permite mover el elemento hacia el frente (en el eje Z). Es como un modelado de capas.

Una vez definidas las propiedades TOP Y LEFT ya no hace falta definir las demás ya que estas son las propiedades prioritarias.

Position: relative

El valor relative establece que un elemento podrá moverse en cualquier dirección respetando el espacio inicial que se definió. Es decir que puede moverse libremente sin afectar a los demás elementos, ninguno de los demás va a modificar su ubicación y se mostrará en la página siempre y cuando este sobre los demás en cuanto al eje Z.

Position: absolute

El valor absolute define a un elemento SIN LUGAR. Es decir el elemento que tenía su ubicación en la página web, con esta propiedad ya no la tiene dejando así a los demás elementos ocupar su ubicación. El elemento aun asi sin tener ubicación se va a seguir mostrando en la página pero este dependerá de su valor en el eje Z. Utiliza como punto de referencia a sus contenedores relativos.

Position: fixed

El valor fixed define a un elemento como fijo en una ubicación predeterminada de la ventana. Este elemento deja de tener su espacio asignado y se desplaza en relación a la ventana gráfica.

POSITION: FIXED
