Multimedia

En html existen etiquetas para poner en nuestra página web archivos de multimedia, ya sean imagenes, videos o audios.

Imágenes

Como se ve, a través de la etiqueta "img" podemos agregar imágenes o GIFS a nuestra página web y enlazamos la misma mediante el atributo "src".

Vídeos

La isla bonita - Madonna.mp4

Para agregar videos de nuestra pc a la página web debemos de utilizar la etiqueta "video" donde agregaremos los siguientes atributos:

  • src

    • Para seleccionar el video en nuestra pc. Recomendable seleccionar un video en formato MP4

  • controls

    • Para que el usuario pueda interactuar con el vídeo.

    Atributos adicionales:

  • Autoplay

    • Reproduce automáticamente el video.

  • Loop

    • Reproduce el video en bucle.

  • Poster

    • Especifica una imagen de portada para el video. Esta se mostrará antes de reproducirse y si el video no puede ser cargado.

  • Muted

    • Reproduce el video sin sonido

Audios

Para agregar audios a nuestra página web es necesario utilizar la etiqueta "audio" acompañada de los siguientes atributos:

  • src

    • Para seleccionar el audio en nuestra PC. Es recomendable que el audio esté en formato MP3 ya que tiene un mejor soporte.

  • controls

    • Para que el usuario pueda interactuar con el audio.

  • autoplay

    • Para que el audio se inicie automaticamente ni bien la página se inicia.

  • muted

    • Para que el audio este muteado.

  • loop

    • Reproducir el audio en bucle.
