Formularios

Los formularios sirven para pedirle al usuario distintos tipos de datos como pueden ser nombre, email, contraseña, etc.

Crear un formulario

Para crear un formulario debemos primero utilizar la etiqueta "form" y dentro de esta agregar los tipos de formularios que querramos utilizar mediante la etiqueta "input" acompañada de un valor "type" para seleccionar el tipo de dato que vamos a ingresar.

Tipos de formularios

    Input type button

    • Es un botón sin comportamiento específico.

    Input type submit

    • Este input sirve como botón para enviar la información.

    Input type text

    • Este formulario sirve para ingresar datos de tipo texto.

    Input type email

    • Formulario para ingresar un Email. Corrobora que tenga el formato palabras@palabras.com

    Input type password

    • Input para ingresar contraseñas.

    Input type checkbox

    • Es una casilla de selección. Se debe utilizar el atributo "value" para definir el valor que se enviará por este elemento. Utilizamos el valor "checked" para definir como ya marcado automaticamente.

    Input type Date

    • Registra una fecha.

    Input type File

    • Permite al usuario subir un archivo desde su computadora.

    Input type Number

    • Permite al usuario registrar un número y solamente un numero.
