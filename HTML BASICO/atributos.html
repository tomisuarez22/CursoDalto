Atributos

Los atributos nos sirven para determinar información adicional a los elementos HTML sobre como se debe procesar o mostrar el contenido.

Entre los atributos de HTML podemos destacar:

  • alt

    • El atributo alt debe estar en los elementos para que el motor de búsqueda se guíe logrando asi una mejor semántica en la página.

  • href

    • Define un vínculo/recurso que debe vincularse en la etiqueta a.

  • target

    • Se emplea para abrir un enlace en un destino distinto a la ventana actual.

  • style

    • Permite definir como se ven algunos elementos de una página web.

  • rel

    • Describe la relación que mantiene el documento destino con el documento presente.

Atributos en formularios

Los formularios requieren ciertos atributos para asi manejarnos de una manera más profesional y para facilitar tambien a la hora de recibir los datos en un servidor.

  • Placeholder

    • Texto que podemos colocar dentro del input que nos sirve como guia sobre el dato que ese input requiere.

  • required

    • Atributo que obliga al usuario ingresar si o si el dato en cierto input.

  • value

    • Coloca un valor determinado ya escrito en el input.

  • name

    • Valor que sirve como referencia a la hora de recibir los datos en el servidor.
